# FakeHack
Фейковый взлом. Обязательно прочитать README

Фейк Взлом якобы узнает почту жертвы

Фейковый взлом. Что бы изменить название приложения найдите root.title("FakeHack") и измените FakeHack на любое название. Инструкцию по использованию: скачайте Python с оригинального сайта. Скачайте Visual Studio Code. Создайте новый Пайтон файл и вставьте этот код. После запуска введите номер телефона жертвы (взлом ненастоящий). Нажмите найти и в терминале введите почту своей жертвы. После ввода перейдите в приложение которое открылось и откройте приложение на весь экран и готово.
